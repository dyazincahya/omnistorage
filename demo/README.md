# OmniStorage Interactive Demo

This folder contains a browser-based interactive playground for manually testing OmniStorage-like storage operations.

## How to use

Open `demo/index.html` in a browser, then choose a storage engine and click the available operation buttons.

Supported demo engines:

- Memory
- LocalStorage
- SessionStorage
- IndexedDB

The demo is intentionally browser-only and does not use Node-only engines such as file storage or server-side SQLite.
